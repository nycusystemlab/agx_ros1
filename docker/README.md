# 🚀 AGX ROS 1 Dashboard 使用指南

這是一個**零依賴**的 Web 控制面板，專為 AGX 機器人開發設計。它能讓你透過網頁直接控制 Docker 服務、啟動 ROS 任務，甚至可以用虛擬搖桿遠端開車。

## 🛠️ 快速啟動

1. **進入目錄**：
   ```bash
   cd agx/ros1_ws
   ```

2. **啟動面板**：
   ```bash
   python3 dashboard.py --port 8090
   ```

3. **訪問網頁**：
   開啟瀏覽器，輸入 `http://localhost:8090` (或者機器人的 IP 位址)。

---

## 📖 功能說明

### 1. 核心服務 (Docker Services)
顯示目前的 Docker 容器狀態。
- **▶ Start**: 啟動該單一服務。
- **⏹ Stop**: 停止並移除該容器。
- **📋 Logs**: 查看該容器的後台日誌。
- **全部啟動/停止**: 一鍵管理整個 ROS 1 棧。

### 2. 自動化任務 (Tmux Tasks)
這些是預設好的 ROS 指令，會在容器內部的 Tmux 視窗執行。
- **▶ 啟動**: 在後台執行 ROS 指令。
- **📋 Log (重要)**：點擊後可以**即時查看終端輸出**，如果程式崩潰或報錯，這裡就是你的 Debug 神器。
- **⏹ 停止**: 關閉該任務的 Tmux Session。

### 3. 🎮 虛擬搖桿
不需實體手把，直接網頁開車。
- **操作方式**：滑鼠點擊按鈕，或使用鍵盤 **W/A/S/D**。
- **急停**：按下 **空白鍵** 或 **⏹ 停止**。
- **速度控制**：拉動拉桿即可即時調整移動速度。

---

## 🛠️ 開發與自定義
如果你想修改任務指令或增加新按鈕，只需編輯 `dashboard.py` 開頭的 `SERVICES` 與 `TASKS` 字典即可，修改後不需編譯，重啟腳本立即生效！
